This is an example calendar built with d3.js. The data set is random and the calendar layout is inspired by Rick Wicklin and Robert Allison.

The calendar supports a number of features where most can be disabled simply via css.

  • Year, month, day of the week labels
  • Tooltips for the days
  • Title with description dropdown panel
  • ColorBrewer scales (set/reset on the svg container)
  • Legend for the color scale
  • Automatic resize to the container (can be disabled)
