Scatter Demo

This is a reusable scatter plot produced with d3.js. The scatter plot allows you to compare, up-to, five variables; x, y, z, color and the size/radius. The plot grammar should be fully responsive however it does not yet add/remove marks as the size gets large/small for them. Here are some features of the chart:

  • a responsive, reusable d3 chart
  • Zoom in/out on the x/y axis
  • Change the x/y axis scale and dimension
  • X vs. Y vs. Z vs. Color vs. Size
  • Changeable aggregate function as well as the four quadrants
  • Interactive z-dimension (hover over z-axis overlay)
  • Object constancy on transitions
